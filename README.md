## 六、Plan Mode 说明

| 模式 | 说明 |
|------|------|
| **Mission Mode** | PRD → 确认 → 执行，适合复杂任务 |
| **Plan Mode** | 你给一个方向，我持续跟进、提醒、推进 |
| **区别** | Mission 是一次性，Plan 是持续性 |

**Plan Mode 工作方式**：
1. 你定义方向和优先级
2. 我定期（每周/每两周）汇报进展
3. 我提醒你待办、跟进、调整
4. 你可以随时暂停、调整、取消
